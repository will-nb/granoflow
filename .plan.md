# 优化月视图链接样式并取消日历点击跳转

## 任务概述

1. **优化月视图链接样式**：将"当日专注时长"和"当日任务"两个统计卡片的文字和数字显示为主题定义的链接颜色（不带下划线），作为链接使用，移除链接图标
2. **取消日历点击跳转**：移除月视图中点击日期跳转到日视图的功能

## 修改文件

### 1. `lib/presentation/review/widgets/month_detail_view.dart`

**修改内容**：

修改 `_StatCard` 组件，当 `onTap` 不为 null 时，将文字和数字显示为主题链接颜色，移除链接图标。

**具体修改**：

1. 修改 `_StatCard` 的 `build` 方法：
   - 如果 `onTap` 不为 null，将 `value` 和 `label` 的 `Text` widget 的颜色设置为 `theme.colorScheme.primary`（主题链接颜色）
   - 移除右下角的 `Positioned` 图标
   - 保持 `InkWell` 包装以支持点击交互

### 2. `lib/presentation/review/calendar_review_page.dart`

**修改内容**：

移除 `onDaySelected` 回调中的跳转逻辑，只保留日期选择功能。

**具体修改**：

1. 在 `_buildLandscapeLayout` 方法中（约第184行）：
   - 移除 `onDaySelected` 回调中检查 `hasCompletedTasks` 的逻辑
   - 移除 `setViewMode(CalendarViewMode.day)` 和 `loadDailyData` 调用
   - 只保留 `selectDate(selectedDay)` 调用

2. 在 `_buildPortraitLayout` 方法中（约第309行）：
   - 同样移除跳转逻辑，只保留日期选择功能

## 预期效果

- 月视图中"当日专注时长"和"当日任务"的文字和数字显示为主题链接颜色（如蓝色），不带下划线
- 点击这两个统计卡片会跳转到日视图
- 点击日历日期不再跳转到日视图，只更新选中日期
- 链接样式更简洁美观，与主题协调

## 测试要点

1. 验证月视图中链接文字和数字显示为主题链接颜色
2. 验证链接没有下划线
3. 验证点击日历日期不会跳转到日视图
4. 验证点击统计卡片中的链接能正确跳转到日视图
5. 验证链接样式美观且与主题协调
